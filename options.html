LIT-SYNTH Kurulumu

Nasıl API Anahtarı Alınır?
  1. Google AI Studio sayfasına gidin.
  2. Sağ üstteki "Create API key" butonuna tıklayın.
  3. "Name your key" sütununa "LIT-SYNTH" yazın.
  4. "Choose an imported project" -> "Create New Project" -> "LIT-SYNTH" -> "Create Project" aşamalarını takip edin.
  5. Gelen kutucukta "Create Key" deyin ve ekranınızdaki Key sütununun altındaki (örn: ...1xBc) yazısına tıklayıp açılan yerden API Key'i kopyalayın.
  6. Kopyaladığınız kodu aşağıdaki kutuya yapıştırın.

✅ Başarıyla Kaydedildi!
